# Scoop Bucket for Autocommit

This is a [Scoop](https://scoop.sh/) bucket for [autocommit](https://github.com/jsmenzies/autocommit), an AI-powered git commit message generator.

## Installation

```powershell
# Add the bucket
scoop bucket add autocommit https://github.com/jsmenzies/scoop-autocommit

# Install autocommit
scoop install autocommit
```

## Updating

```powershell
scoop update
scoop update autocommit
```
